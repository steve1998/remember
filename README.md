# Remember
Chrome extension to keep track of important things such as links, notes, etc.
This app uses the boilerplate made by: https://github.com/satendra02/react-chrome-extension

# Current Features
Current features that are available are:
- Creating links
- Deleting all links

Future features to be implemented:
- Implement a backend server that stores users' data and a login feature for users
- Deleting a single link
- Updating the design
